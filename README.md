# Neopets Quickstock Restyler

> Makes some simply styling tweaks to the redesigned quickstock page to make it easier to use and more user friendly.

---

## 📦 Installation

### 1. Install Tampermonkey

To run this userscript, you’ll need a userscript manager extension:

* Install **Tampermonkey** for your browser:

  * Chrome / Edge: https://www.tampermonkey.net/
  * Firefox: https://addons.mozilla.org/firefox/addon/tampermonkey/

---

### 2. Install the Script

#### Option A: Install from file

1. Open Tampermonkey dashboard
2. Click **"Create a new script"**
3. Replace the default content with your script code
4. Press **Ctrl + S** (or File → Save)

---

#### Option B: Install from URL (if hosted)

1. Open the raw script URL (e.g. from GitHub)
2. Tampermonkey should automatically detect it
3. Click **Install**

---

## 🚀 Usage

1. Navigate to the quickstock page
2. The script will run automatically
